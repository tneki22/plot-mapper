# Разметка карты участков (Streamlit)

Веб-инструмент для быстрой разметки карты участков и экспорта результата в `plots.json`.

## Что умеет

- Разметка объектов в режимах `Прямоугольник` и `Полигон`.
- Типы объектов: участок, детская площадка, проезд, зона отдыха, парковка, другое.
- Для участка: номер, улица, площадь, статус (свободен/продан/резерв).
- Сохранение каждого объекта в сессионный список.
- Быстрое исправление разметки: удаление последней несохраненной фигуры или очистка всех несохраненных фигур.
- Откат последнего объекта и удаление любого объекта из таблицы.
- Экспорт JSON по контракту для Django-импорта.
- Импорт ранее сохраненного JSON для продолжения работы.
- Координаты сохраняются в системе исходной карты (без зависимости от масштаба отображения).

## Локальный запуск

1. Откройте папку проекта:

```bash
cd plot-mapper
```

2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Запустите приложение:

```bash
streamlit run app.py
```

4. В браузере откроется интерфейс разметки.

## Деплой в Streamlit Community Cloud

1. Создайте GitHub-репозиторий `plot-mapper`.
2. Запушьте содержимое папки в репозиторий.
3. Зайдите на https://share.streamlit.io.
4. Нажмите `New app` и выберите:
   - Repository: ваш `plot-mapper`
   - Branch: `main`
   - Main file path: `app.py`
5. Нажмите `Deploy`.
6. Через несколько минут получите публичный URL вида `https://...streamlit.app`.

## Формат экспорта JSON

Файл `plots.json`:

```json
{
  "image": "plan.jpg",
  "image_size": {"width": 717, "height": 1183},
  "features": [
    {
      "type": "plot",
      "number": 15,
      "street": "Ивана Стрешнева",
      "area_m2": 10.5,
      "status": "free",
      "polygon": [[123, 456], [180, 456], [180, 510], [123, 510]],
      "note": ""
    },
    {
      "type": "playground",
      "label": "Детская площадка",
      "polygon": [[300, 900], [350, 900], [350, 950], [300, 950]],
      "number": null,
      "street": null,
      "area_m2": null,
      "status": null
    }
  ]
}
```

## Рекомендации по командной работе

- Разделите карту на зоны и назначьте каждую зону отдельному человеку.
- Каждый выгружает свой JSON.
- После разметки объедините JSON-файлы в один `plots.json`.

## Быстродействие

- В приложении отключены лишние realtime-обновления холста.
- Карту можно масштабировать только для удобства работы, экспорт всегда идет в исходных координатах.
- При большом числе объектов сохраняйте работу по зонам (например, по улицам), это ускоряет ручную проверку.
